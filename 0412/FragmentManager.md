# Fragment Manager

프래그먼트 안에 프래그먼트를 만들었다면?

`FragmentA` in `Activity` : Support Fragment Manager
`FragmentB` in `FragmentA` : Child Fragment Manager
`FragmentA` can call `FragmentB` : Parent Fragment Manager
